## Overview

Welcome to the AI for Medical Imaging workshop! This repository contains the code, data pointers, and instructional notebooks used during the session. The workshop focuses on applying deep learning techniques—specifically Convolutional Neural Networks (CNNs)—to real-world medical imaging tasks.

## Contents

- **`Starter_code.ipynb`**: Initial code provided to participants to follow along with the workshop exercises.
- **`Vibe_Coding.ipynb`**: Supplemental or completed notebook demonstrating the concepts discussed during the session.
- **`generate_nb.py`**: A utility script related to generating or managing notebook files for the workshop.

## Getting Started

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/your-username/SDSS_Medical_Imaging_Using_CNN.git
   ```
2. Open the Jupyter Notebooks:
   If you have Jupyter installed, you can launch it in the repository directory:
   ```bash
   jupyter notebook
   ```
   Alternatively, you can upload the notebooks to Google Colab for an environment with pre-configured GPU support.

## Prerequisites

Basic familiarity with Python and concepts in machine learning will be helpful. We recommend using Google Colab for the exercises to ensure everyone has access to the necessary computational resources without complex local setup.
